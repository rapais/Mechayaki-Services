# Mechayaki - Street Food Pre-Order System

Full stack interview demo:
- Backend: Java Spring Boot 3 (REST, JWT, PostgreSQL)
- Frontend: React + Vite

## Run locally (dev)

1)  Start DB:
    docker compose up -d

2)  Backend:
    cd backend
    nvm spring-boot:run
    
3)  Frontend:
    cd frontend
    npm install
    npm run dev
